# blogNode

npm install  

Pm2 start 
npm run start-prod

Local start
npm start  
  
project structure  
    .  
    ├── config                  # Configuration file for "config-lite"  
    ├── lib                     # Database design  
    ├── middlewares             # Middlewares  
    ├── models                  # The files that hold the operating database  
    ├── public                  # Static files  
    ├── routes                  # Route files  
    ├── views                   # Template file  
    ├── app.js                  # Express configuration   
    ├── ecosystem.config.js     # pm2 configuration   
    └── www                     # The entrance
